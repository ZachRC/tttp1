API Documentation

Authentication

Secure your API requests using JWT tokens. All requests must include an Authorization header.

Authorization: Bearer your-jwt-token

Endpoints

GET /api/v1/accounts

Retrieve all connected TikTok accounts.

POST /api/v1/comments/create

Create a new automated comment rule.

PUT /api/v1/settings/update

Update automation settings for an account.
